Tecnologías Web 2022.

Las tecnologias web han evolucionado durante las ultimas decadas, hoy en día existen una multitud de aplicaciones, frameworks, arquitecturas, librerías y sistemas que nos proporcionan una amplia gama de soluciones para la gestion de paginas, sitios y portales web.
Por lo que Tecnologias-web brindará los conocimientos y competencias necesarias para ser desarrolladores frontend, backend y fullstack.

Unidades de Aprendizaje

Introduccion a las Tecnologías WEB

En este Apartado aprenderas conceptos de NTIC, definiciones web, además de HTML 5, CSS, JAVASCRIPT, PHP y mucho más

Solución Tecnologica WEB

Aprenderas fundamentos de JAVASCRIPT, uso de Github, bitbucket, DOM, CMS, y conceptos de frontend y backend

CMS y Frameworks

Aprenderás Node,js, y uso de Frameworks como son VUE y Angular.
